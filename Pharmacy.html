Online Pharmacy Store Project

Java J2EE E-Pharma Portal with SQL Integration "A robust web application built to streamline pharmacy operations. Leveraged Java J2EE for server-side logic and JDBC for seamless communication with a MySQL database. Implemented secure session management for user login/registration. The backend architecture supports real-time product updates by the Admin and a shopping workflow for users, ensuring data integrity and a smooth transactional experience."
